Various level packs stored with one level per line.
These are alredy labeled, and can be prepared to be dropped into the game using `runner.py package`.
They can also be copied over to game/puzzles.txt to view them in game/viewer.html.
Each line is valid JSON and so files can be parsed one line at a time.

generated.txt: The generated set of puzzles used in the Infinite Refraction study.
original.txt: The original game, with refraction 2 rules (no coins, converters)
brainpop.txt: The default brainpop progression circa fall 2013.

Original and Brainpop are in the order puzzles appeared in their respective progresions.
Generated is in an arbitrary order, the progression was dynamically generated.
